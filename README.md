# Classic Snake Game

Classic Snake is a simple and fun arcade-style game inspired by the legendary Snake game.
Control the snake using the arrow keys and try to grow as long as possible without hitting the walls or yourself.

🎮 Controls

⬆️ Arrow Up — Move Up

⬇️ Arrow Down — Move Down

⬅️ Arrow Left — Move Left

➡️ Arrow Right — Move Right

⭐ Features

Classic snake gameplay

Easy-to-use arrow key controls

Increasing difficulty as the snake grows

Clean and minimal design

Perfect for learning, practicing logic, or just having fun! 🕹️🐍
